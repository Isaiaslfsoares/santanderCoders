# Aula 05 - Exercicio em grupo #1:

Crie um jogo da adivinhação que gere um número aleatório entre 1 e 100 e permita que o jogador tente adivinhar, seguindo os seguintes requisitos:

1. O jogador tem no máximo 10 palpites.
2. O jogo deve solicitar novos palpites enquanto o jogador não acertar e ainda houverem palpites disponíveis.
3. Caso o jogador acerte o número, exiba uma mensagem de sucesso.
4. Caso os palpites se esgotem, exiba uma mensagem de "Game Over" e informe qual era o número sorteado.
5. Ao inserir um palpite errado o jogo deve informar se o valor é maior ou menor que o último palpite do usuário quando solicitar o próximo palpite.

```javascript
// Exemplo de função para geração de número aleatório:

function gerarNumeroAleatorio(min, max) {
  return Math.floor(Math.random() * (max - min + 1) + 1);
}
```
